# Developer Docs

Prereqs:

Node, gulp, browserify, watchify
- https://www.typescriptlang.org/docs/handbook/gulp.html

Installation steps:

`git clone https://github.com/visdesignlab/data-forensics.git`

`cd data-forensics/`

`npm install`

To build/run:
`gulp`
